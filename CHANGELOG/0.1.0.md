# stopwatch 0.1.0

Initial release.

## Added

- Integer-nanosecond `Duration` parsing, formatting, conversion, arithmetic, and comparison.
- Strict `Stopwatch` lifecycle with pauses, laps, injected clocks, sync/async contexts, and immutable measurements.
- Sync, async, generator, and async-generator decorators.
- Thread-safe nested `TimerRegistry` aggregation, bounded retention, percentiles, filtering, and reports.
- Performance budgets, JSON/CSV exports, built-in sinks, sampling, and runtime configuration.
- Dependency-free command-line tools for formatting, parsing, command timing, and report display.

## Compatibility

- Python 3.12 and later.
- MIT licensed.
